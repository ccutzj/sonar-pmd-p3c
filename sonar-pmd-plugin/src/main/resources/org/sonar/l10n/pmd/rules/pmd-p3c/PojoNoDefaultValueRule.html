定义DO/DTO/VO等POJO类时，不要加任何属性默认值.

不合规代码示例

class MyClass{
  private Long id;
  private String unitAccnum;
  private String dwzhzt = OPTION_CODE_STATUSTYPE_NORMAL; // Noncompliant
  private String jzny;
}

合规解决方案

class MyClass{
  private Long id;
  private String unitAccnum;
  private String dwzhzt;
  private String jzny;
}
