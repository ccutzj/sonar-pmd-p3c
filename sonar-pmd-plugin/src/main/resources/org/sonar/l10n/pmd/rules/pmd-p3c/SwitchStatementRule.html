在一个switch块内，每个case要么通过break/return等来终止，要么注释说明程序将继续执行到哪一个case为止；在一个switch块内，都必须包含一个default语句并且放在最后，即使它什么代码也没有.

不合规代码示例

class MyClass {
  public void BadMethod() { }
}

合规解决方案

class MyClass {
  public void GoodMethod() {
    switch (str) {
      case "0":
        // ...
        break;
      case "1":
        // ...
        break;
    default:
        break;
    }
    // 注意 break 是退出 switch 语句块，而 return 是退出方法体
  }
}
