方法内部单行注释，在被注释语句上方另起一行，使用//注释。方法内部多行注释使用/* */注释。注意与代码对齐.

示例

    public enum TestEnum {
        /**
         * agree
         */
        AGREE("agree"),
        /**
         * reject
         */
        REJECT("reject");

        private String action;

        TestEnum(String action) {
            this.action = action;
        }

        public String getAction() {
            return action;
        }
    }
