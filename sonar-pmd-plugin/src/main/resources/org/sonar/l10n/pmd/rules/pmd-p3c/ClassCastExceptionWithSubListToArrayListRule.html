ArrayList的subList结果不可强转成ArrayList，否则会抛出ClassCastException异常.

subList 返回的是 ArrayList 的内部类 SubList，并不是 ArrayList 而是 ArrayList 的一个视图，对于 SubList 子列表的所有操作最终会反映到原列表上。

不合规代码示例

class MyClass {
  List<BizBatchData> bizBatchData = bizBatchDataList.subList(0, 10);
  bizBatchData = (ArrayList)bizBatchData;
}
