# ShortClassName
**Category:** `pmd`<br/>
**Rule Key:** `pmd:ShortClassName`<br/>
> :warning: This rule is **deprecated** in favour of [S101](https://rules.sonarsource.com/java/RSPEC-101).

-----

Classnames with fewer than five characters are not recommended. Example:
<pre>
public class Foo {
}
</pre>
